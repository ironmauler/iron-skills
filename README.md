# iron-skills

Personal Claude skill bundle — writing, blog pipeline, design, and frontend tools.
A Claude Code / Cowork plugin marketplace.

## Skills (52)
stop-slop, the full blog pipeline (30 skills), design-taste-frontend, high-end-visual-design,
minimalist-ui, industrial-brutalist-ui, redesign-existing-projects, image-to-code,
imagegen (web/mobile), brandkit, React/Vercel helpers, and more.

## Add to Claude Code (terminal)
```
claude plugin marketplace add ironmauler/iron-skills
claude plugin install iron-skills@iron-skills
```

## Add to Cowork
Desktop app → plugin / marketplace settings → add marketplace:
`https://github.com/ironmauler/iron-skills`
then enable the **iron-skills** plugin.
